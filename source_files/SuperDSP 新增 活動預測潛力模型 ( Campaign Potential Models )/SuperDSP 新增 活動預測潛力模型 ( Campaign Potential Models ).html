新增「活動預測潛力模型」抽屜

開發項目 / 需求 新增「活動預測潛力模型」抽屜
系統 SuperDSP
模組 Ad Group (廣告組合)
頁面 / 元件 Ad Group 新增/編輯抽屜 > 數據鎖定
CommerceAdDrawer.vue
Ticket Number  
開發人員 alice
預估時間 2D
基本資訊 開啟方式：Ad group 點擊「新增 / 編輯」按鈕 → 數據鎖定 → 點擊「活動預測潛力模型」
抽屜標題：數據鎖定 - 活動預測潛力模型

抽屜組成

  • 可選受眾包區域 (左)：顯示可選擇的受眾包清單，提供篩選功能
  • 已選受眾包區域 (右)：顯示當前已選取的受眾包項目，提供清除和取消選取功能

受眾包項目資訊

  • Audience Pack Name
  • 狀態 Icon + Tooltip（依據狀態規則顯示）
備註
  • 僅支援廣告品類：CommerceAd
  • 在後端 API 回傳資料中，將 Campaign Potential Models 歸屬在 segment_ids 裡

可選受眾區域

篩選功能
  • 可依 Name 篩選
  • 無符合條件時：顯示空白狀態 +「無資料」提示
選擇行為

選中項目後：

  1. 從左欄移除
  2. 出現在右欄「已選受眾」區域
  3. 檢查數量限制，達到上限時變更為禁用狀態

已選受眾區域

數量顯示

根據已選項目數量即時變化顯示。

移除方式
  • 清除按鈕：清空所有已選項目
  • 單項取消：
    • 點擊已選項目
    • 該項目回到左欄「可選受眾」區域

排序規則

  • 按 ID 由新到舊（大 → 小），不受停用狀態影響

狀態顯示

狀態 Icon 與 Tooltip

根據受眾包狀態顯示對應的 Icon 與 Tooltip：

狀態 Icon Tooltip 訊息
UU = 0 ⓘ 無有效受眾

選取數量限制

區域 受眾包來源 限制數量
Include 特定族群、鑽石會員受眾、產業 Shopper Graph、Webb 零售數據、廣告活動受眾包、網站流量受眾包、客製受眾包、全域預測潛力模型 合計最多 100 個

超過限制時：可選區域選項狀態變更為 disabled，無法繼續選取。
