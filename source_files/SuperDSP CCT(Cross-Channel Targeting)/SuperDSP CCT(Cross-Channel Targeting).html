1. SuperDSP CCT(Cross-Channel Targeting) 產品規格

原始頁面：https://guoshi.atlassian.net/wiki/spaces/ITO/pages/3997433857/SuperDSP+CCT+Cross-Channel+Targeting
最後修改：2025-09-08

緣由

  • CCT 是一個獨立物件，類似 One Pixel，擁有自己的資料結構。
  • 目的是將資料同時打到不同廣告平台，提升投放精準度與效果。

目的

  1. 提供統一跨渠道目標設定介面，方便管理與追蹤。
  2. 支援多平台 Pixel ID 管理，提升數據追蹤準確性。
  3. 支援多語言介面。

User Story

  • 建立/管理 CCT Pixel，維護可重複使用的追蹤資源。
  • 檢視已建立的 Pixel。
  • Ad Group 建立/編輯時綁定 Pixel。
  • 管理者可於「設定」管理 CCT 權限。

Wireframe

  • PPT by Helen

主要流程摘要

  1. 建立 Cross Channel Pixel：選擇平台、輸入 Pixel ID、設定事件類型與名稱，儲存後自動產生初始碼。
  2. 編輯 Pixel：可編輯 Pixel ID 與事件設定，Channel 不可更改。
  3. Ad Group 綁定 Pixel：選擇要關聯的 CCT Pixel，系統檢查格式要求，完成後更新設定。

2. 前端開發文件

原始頁面：https://guoshi.atlassian.net/wiki/spaces/ITO/pages/4007329797
最後修改：2025-09-26

User Stories & Acceptance Criteria

  • 建立/編輯/檢視 Pixel，權限控管（詳見角色權限矩陣）
  • Ad Group 綁定 Pixel，支援選擇/更換/移除
  • 管理者可於「設定」管理 CCT 權限

角色權限矩陣

功能 / 角色 super_admin agency_admin / client_admin ad_operator read_only brand_client media
檢視 Pixel 列表 ✅（可切換查看所有代理商/直客/媒體帳號） ✅ ✅ ✅ ✅ ❌
新增 Pixel ❌（僅輔助檢視與排查，不建立資源） ✅ ✅ ❌ ❌ ❌
編輯 Pixel ❌ ✅ ✅ ❌ ❌ ❌

主要前端功能

  • Cross Channel 列表頁：搜尋、分頁、權限判斷、i18n
  • 新增/編輯 Pixel 抽屜：表單驗證、動態 placeholder、事件名稱規則、API 整合
  • Ad Group 綁定 Pixel：支援多選、radio、未指定選項、查無資料提示
  • 設定頁面：進階設定管理 CCT 權限

表單驗證規則

  • Name：必填，80字內，去除前後空白
  • Channel：必填，select，依字母排序
  • Pixel ID：必填，依 Channel 動態 placeholder，去除前後空白
  • Advertising Category & Format：必填，select
  • Trigger Event Type：多選，依 API 指定順序
  • Event Name：必填，僅允許英數字與底線，英文字母開頭，長度依 Channel 限制

3. SuperDSP CCT API

Channel (渠道) 選項

GET /api/v1/superdsp/options/cct_channels
{
  "success": true,
  "data": [
    {
      "id": 1,
      "name": "Google Ads",
      "tracking_id_placeholder": {
        "en": "ID without prefix (e.g. AW-123456789: 123456789)",
        "zh": "無前綴，範例：AW-123456789輸入123456789"
      },
      "event_name_length_limit": 40
    },
    ...
  ]
}

Trigger Event Types (觸發事件類型) 選項

GET /api/v1/superdsp/options/cct_event_types
{
  "success": true,
  "data": [
    {
      "id": 1,
      "name": { "en": "Impression", "zh": "曝光" },
      "default_value": "superdsp_impression",
      "cct_product_category_ids": [1,2,3]
    },
    ...
  ]
}

建立/更新/查詢 Cross Channel Pixel

POST /api/v1/superdsp/cct_accounts
PUT  /api/v1/superdsp/cct_accounts/${id}
GET  /api/v1/superdsp/cct_accounts/${id}
GET  /api/v1/superdsp/cct_accounts?company_id=${company_id}

Product Category 選項

GET /api/v1/superdsp/options/cct_product_categories
{
  "success": true,
  "data": [
    { "id": 1, "name": "InStream" },
    { "id": 2, "name": "RichMedia Display Format" },
    { "id": 3, "name": "RichMedia Video Format" }
  ]
}

Ad Group 綁定 CCT

POST /superdsp/campaigns/${campaign_id}/ad_groups
PUT  /superdsp/ad_groups/${ad_group_id}
GET  /superdsp/ad_groups/${ad_group_id}

驗證規則

  • event_names.value 僅可輸入英數字與底線，且必須以英文字母開頭
  • 不同 channel 對應不同的字數限制
  • 同一 channel 只能有一個 pixel id
