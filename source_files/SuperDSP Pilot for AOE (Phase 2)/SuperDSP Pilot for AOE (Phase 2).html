### 無資料時報表顯示說明
從 ClickHouse 取得的原始數據中，無資料預設值為 `0`。 
但在報表呈現時，依據指標特性會區分為 `0` 或 `-` 兩種顯示方式，來代表不同的語意。

### 指標分類及規則
| 分類 | 定義 | 說明 |
| --- | --- | --- |
| 一階指標 | 直接從後端取得的原始數據 | 不需計算，直接呈現 |
| 二階指標 | 經過公式計算產生的數據 | 有分子/分母關係 |
| 指標類型 | 分類 | 無資料時顯示 | 原因 |
| --- | --- | --- | --- |
| 曝光 | 一階 | `0` | 基礎指標，投放即產生紀錄 |
| 點擊、轉換等互動指標 | 一階 | `0` | 計數類指標，0 = 沒有發生 |
| 金額類（花費、收益、購物車金額等） | 一階 | `$0` | 金額類指標，保持格式一致 |
| CTR、CPM 等比率/成本指標 | 二階 | `-` | 分母為 0 時無法計算 |
二階指標補充說明
二階指標的計算公式皆有分母：當分母 = 0 時，計算結果為無效值（無法除以零），顯示 `-` 表示「不適用 / 無法計算」，避免誤導使用者認為成本或效率為零。
| 指標 | 公式 |
| --- | --- |
| CPM | 花費 / 曝光 × 1000 |
| CPV | 花費 / 30秒完整觀看 |
| 點擊率 | 點擊 / 曝光 |
| 每次進站成本 | 花費 / 進站次數 |
| 每次註冊完成成本 | 花費 / 註冊完成次數 |
| 每次加入購物車成本 | 花費 / 加入購物車次數 |
| 每次開始結帳成本 | 花費 / 開始結帳次數 |
| 每次完成購買成本 | 花費 / 完成購買次數 |
| 廣告投報率 (ROAS) | 收益 / 花費 |
| 觀看率 (25%/50%/75%/100%) | 觀看次數 / 曝光 |

- 針對「廣告組合報表」下載的 Excel 檔案，統一並明確定義各數值欄位的呈現格式與無效欄位處理規則。

- 技術重點：Excel `numFmt` 格式變體、`columnConfig` 更新、`` 文件同步。

## 數值格式細分與顯示規則
- 所有數值欄位的顯示格式，依據對應的 Excel `numFmt` 定義處理。
- 當數值為 **0** 時，且該欄位格式為 `_DASH` 類型，顯示為 `-`。
- 當數值為 **null / undefined / 無資料** 時，視為無效值，依「無效欄位處理規範」處理。

### 指標格式套用規則
- 以下指標依既有定義套用 `_DASH` 類型格式：
  - `cpm`
  - `cpv`
  - `ctr`
  - `vtr25` ~ `vtr100`
  - `cost_per_*`
  - `roas`
- 上述指標於數值為 0 時，顯示為 `-`，其餘數值依 `numFmt` 顯示。

## Excel 格式規則更新
- 更新 `columnConfig` 設定，修正各欄位與對應格式。
- 確保貨幣 (Currency) 與百分比 (Percentage) 欄位具備正確的小數點與單位顯示。

## 無效欄位處理規範
- 無效欄位定義：當欄位因廣告格式或資料條件限制而無法計算（例如 Banner 格式無四階段觀看率），該欄位視為無效欄位。
- 後端回傳規則：後端針對無效欄位，統一回傳字串 `"-"`。
- 前端顯示規則：
  - 前端不需對無效欄位進行任何數值轉換或判斷。
  - 前端需原樣呈現後端回傳值，包含系統畫面與下載報表。

##  資訊同步
- 格式規則紀錄：於文檔中標註數值千分位、整數貨幣、兩位/四位小數貨幣與百分比之格式定義與範例。
- 無效欄位說明：補充「無效欄位處理」說明，列舉 `banner` 格式與相關觀看事件之處理範例。
