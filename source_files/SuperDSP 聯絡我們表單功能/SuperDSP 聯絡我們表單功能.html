📖 項目概述

📌 項目背景

目前 SuperDSP 平台缺乏統一的客戶服務聯絡管道，用戶遇到問題時無法直接透過平台內建功能聯繫客服團隊。為了提升用戶體驗並建立完整的客戶服務流程，需要在平台中新增聯絡我們的表單功能，以利讓相關人員可以追蹤平台維運狀況。

🎯 項目目標

  • 提供用戶友善的問題回報介面，讓使用者可以更便利的聯繫到相關人員
  • 支援中英雙語介面，滿足不同用戶需求
  • 整合現有 UI 元件（抽屜功能），保持介面一致性
  • 實現自動化郵件通知機制，確保問題能順利知會到相關人員

🧩 項目範圍

包含：

  • 聯絡我們表單 UI 開發（使用現有抽屜元件）
  • 表單驗證與錯誤提示機制
  • 中英雙語支援
  • 郵件發送 API 整合
  • 成功提交後的用戶反饋機制

不包含：

  • 後台客服管理系統
  • 工單追蹤功能
  • 檔案上傳功能
  • 即時聊天功能

🧩 功能詳情子頁

詳細互動規格

1. 抽屜開啟

  • 觸發條件： 點擊任何「聯絡我們」入口點
  • 動畫效果： 從右側滑入，背景遮罩淡入
  • 初始狀態： 所有欄位為空，字元計數器顯示 0/100 和 0/1000

2. 表單輸入

  • 標題欄位：
    • 單行文字輸入框
    • 即時字元計數，超過 100 字元時輸入框邊框變紅
    • Placeholder：「請輸入問題標題」 / "Please enter subject"
  • 問題描述欄位：
    • 多行文字輸入框，最小高度 120px
    • 即時字元計數，超過 1000 字元時輸入框邊框變紅
    • 支援換行輸入
    • Placeholder：「請詳細描述您遇到的問題」 / "Please describe your issue in detail"

3. 表單驗證

  • 即時驗證： 輸入過程中即時檢查字元數限制
  • 提交驗證： 點擊提交時檢查必填欄位
  • 錯誤顯示： 欄位下方顯示紅色錯誤文字，輸入框邊框變紅

4. 提交處理

  • Loading 狀態： 依照目前其他相同 Loading 顯示
  • 成功處理： 顯示 Toast 訊息，2 秒後自動關閉抽屜
  • 失敗處理： 顯示錯誤 Toast，保持抽屜開啟狀態

🧪 測試用例子頁

功能測試用例

TC001: 成功提交聯絡表單

測試目標： 驗證用戶能夠成功提交聯絡表單

前置條件： 用戶已登入系統

測試步驟：

  1. 從 Avatar 展開選單中點擊「聯絡我們」按鈕
  2. 在標題欄位輸入「測試問題」
  3. 在問題描述欄位輸入「這是一個測試問題的詳細描述」
  4. 點擊「提交」按鈕

預期結果：

  • 顯示載入狀態
  • 出現成功 Toast 訊息
  • 抽屜自動關閉
  • 客服信箱收到郵件

TC002: 必填欄位驗證

測試目標： 驗證必填欄位的驗證機制

測試步驟：

  1. 開啟聯絡表單
  2. 不填寫任何內容，直接點擊「提交」
  3. 只填寫標題，點擊「提交」
  4. 只填寫問題描述，點擊「提交」

預期結果：

  • 步驟2：顯示「標題必填」和「問題描述必填」錯誤訊息
  • 步驟3：顯示「問題描述必填」錯誤訊息
  • 步驟4：顯示「標題必填」錯誤訊息
  • 錯誤欄位邊框變紅色

TC003: 字元數限制驗證

測試目標： 驗證字元數限制功能

測試步驟：

  1. 在標題欄位輸入 101 個字元
  2. 在問題描述欄位輸入 1001 個字元
  3. 觀察字元計數器顯示
  4. 嘗試提交表單

預期結果：

  • 字元計數器正確顯示超出限制
  • 輸入框邊框變紅色
  • 無法提交表單或顯示錯誤訊息

TC004: 多語言切換測試

測試目標： 驗證中英文介面切換

測試步驟：

  1. 在中文介面下開啟聯絡表單
  2. 切換至英文介面
  3. 重新開啟聯絡表單
  4. 測試驗證錯誤訊息語言

預期結果：

  • 所有文字正確切換語言
  • 錯誤訊息使用對應語言顯示
  • Placeholder 文字正確顯示

UI 草圖

  • 參考畫面網址： 聯絡我們

UI 元件規格

  • 觸發方式： 點擊「聯絡我們」按鈕開啟抽屜

  • 標題與問題描述皆為必填
  • 即時字元數統計顯示
  • 錯誤訊息：
    • 中文：「標題必填」、「問題描述必填」
    • 英文：「Subject is required」、「Description is required」

成功提交反饋

  • Toast 訊息：
    • 中文：「我們已收到您的問題，客服人員將盡快與您聯繫。」
    • 英文：「We've received your inquiry. Our support team will contact you shortly。」
