[廣告組合報表] 匯出數值格式優化與無效欄位處理規範

開發項目 / 需求 [廣告組合報表] 匯出數值格式優化與無效欄位處理規範
系統 SuperDSP
Ticket Number ITP-2924
開發人員 Ryan Meng
預估時間 1d
描述 本次調整針對「廣告組合報表」下載的 Excel 檔案，統一並明確定義各數值欄位的呈現格式與無效欄位處理規則。
‧ 影響範圍：廣告組合報表匯出功能 (.xlsx)。
‧ 技術重點：Excel numFmt 格式變體、columnConfig 更新、<spec> 文件同步。
備註 -
功能細項 - 數值格式細分與顯示規則 通用顯示規則
‧ 所有數值欄位的顯示格式，依據對應的 Excel numFmt 定義處理。
‧ 當數值為 0 時，且該欄位格式為 _DASH 類型，顯示為 -。
‧ 當數值為 null / undefined / 無資料 時，視為無效值，依「無效欄位處理規範」處理。

指標格式套用規則
以下指標依既有定義套用 _DASH 類型格式：
‧ cpm
‧ cpv
‧ ctr
‧ vtr25 ~ vtr100
‧ cost_per_*
‧ roas
‧ 上述指標於數值為 0 時，顯示為 -，其餘數值依 numFmt 顯示。
功能細項 - Excel 格式規則更新 ‧ 更新 columnConfig 設定，修正各欄位與對應格式。
‧ 確保貨幣 (Currency) 與百分比 (Percentage) 欄位具備正確的小數點與單位顯示。
功能細項 - 無效欄位處理規範 ‧ 無效欄位定義：當欄位因廣告格式或資料條件限制而無法計算（例如 Banner 格式無四階段觀看率），該欄位視為無效欄位。
‧ 後端回傳規則：後端針對無效欄位，統一回傳字串 "-"。
‧ 前端顯示規則：
  ‧ 前端不需對無效欄位進行任何數值轉換或判斷。
  ‧ 前端需原樣呈現後端回傳值，包含系統畫面與下載報表。
<spec> 資訊同步 ‧ 格式規則紀錄：於文檔中標註數值千分位、整數貨幣、兩位/四位小數貨幣與百分比之格式定義與範例。
‧ 無效欄位說明：補充「無效欄位處理」說明，列舉 banner 格式與相關觀看事件之處理範例。
