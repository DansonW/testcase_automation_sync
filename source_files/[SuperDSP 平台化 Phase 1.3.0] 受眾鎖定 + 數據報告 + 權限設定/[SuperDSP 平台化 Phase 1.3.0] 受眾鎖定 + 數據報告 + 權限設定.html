  廣告組合 (Ad Group) 數據鎖定 - 量體預估
  • 無此功能
  • 預設：0 - 0 UU
  • 計算條件：年齡性別 選取數 > 0
  • 檢查條件並計算時機：
    • 廣告組合抽屜打開時
    • 調整數據鎖定「包含」或「排除」項目，儲存關閉抽屜時
  • 其他：
    • 計算量體 < 10,000 時，顯示警告：
      你的廣告受眾太過特定，可能影響廣告投放，建議將受眾目標設定較廣泛。
      但不阻擋儲存。
    • 呼叫 API 還沒得到結果時，顯示 計算中
 顏色定義：紅色：警告訊息內容
  廣告組合 (Ad Group) 數據鎖定 - 表單區域規則
  • 無此功能
  • 年齡性別：
    • 必填
    • 預設全選
    • 選取數為 0 時，顯示警告 請設定年齡性別。
  • 包含 / 排除 區域：
    • 每個區域分為左中右三個小區域
      • 左：標題
      • 中：被選擇的內容
      • 右：開啟設定抽屜按鈕
    • 預設被選擇的內容只顯示 5 筆資訊，超過顯示「更多」文字按鈕
    • 點擊「更多」按鈕，展開並顯示所有選擇的資訊，文字按鈕變更為「收起」
    • 點擊「收起」按鈕，收合並只顯示 5 筆資訊，文字按鈕變更為「更多」
    • 項目設定後，順序會向上挪，並在項目下方加入 or 分隔線
      • 「已選取項目」跟「已選取項目」之間會顯示分隔線
      • 「已選取項目」跟「未選取項目」之間不會有分隔線
    • 排序規則：
      • 包含：興趣特定議題 ➜ 特定族群 ➜ 產業鑽石 ➜ 廣告活動 ➜ 網站流量
      • 排除：廣告活動 ➜ 網站流量
 顏色定義：紅色：警告訊息內容 或 強調差異用
  廣告組合 (Ad Group) 數據鎖定 - 年齡性別設定
  • 無此功能
  • 分為 男性, 女性 兩部分
  • 標題後方常駐「至少選擇一筆」訊息
  • 接受不選擇任何項目就儲存（廣告組合儲存時才阻擋）
  • 儲存後，會將選擇的項目顯示於「年齡性別」區域中
  
  廣告組合 (Ad Group) 數據鎖定 - [包含]興趣特定議題設定
  • 無此功能
  • 分為 興趣類別, 特定議題 上下兩個區域
  • 區域標題後方顯示「全選」文字按鈕
  • 興趣類別：
    • 選項標籤後方顯示當前興趣下的特定議題選取數量
    • 當特定議題全選時，數量處顯示「(全選)」
    • 點擊「全選」按鈕時，所有「興趣類別」下「特定議題」變為選取狀態，文字按鈕變「清空已選」
    • 若清空或取消任一特定議題，按鈕文字變回「全選」
  • 特定議題：
    • 選項會根據當前選擇的興趣類別變更
    • 點擊「全選」按鈕時，當前興趣類別的所有特定議題變為選取，按鈕文字變「清空已選」
    • 切換到其他興趣類別時，按鈕文字變回「全選」
  • 表單顯示規則：
    • 儲存後，會將選擇的項目顯示於「興趣特定議題」區域中
    • 標籤顯示邏輯：
      • 興趣類別狀態為全選，顯示「興趣類別名稱」
      • 興趣類別非全選，顯示已選擇的「特定議題名稱」
  
  廣告組合 (Ad Group) 數據鎖定 - [包含]特定族群設定 (其他功能)
  • 無此功能

Audience / Segment 抽屜設定共有功能：

  • 分為「可選受眾包」、「已選受眾包」左右兩個區域
  • 可選受眾包 區域：
    • 分為「搜索關鍵字」與「選項區域」上下兩個區域
    • 無選項資料或無符合關鍵字時，選項區域灰底顯示「無資料」
    • 選擇項目後，該項目從「可選受眾包」移到「已選受眾包」
    • 達到選取數限制時，選項變為禁用狀態
  • 已選受眾包 區域：
    • 分為「已選取數量提示 + 清除選取」與「已選擇項目」上下兩個區域
    • 已選取數量隨項目變化
    • 點擊「清除」按鈕，取消所有已選受眾包，項目回到可選受眾包區域
    • 點擊已選擇項目，可取消選取並回到可選受眾包
  • 選取數限制：
    • 包含區域：特定族群 + 鑽石產業受眾 + 廣告活動受眾包 + 網站流量受眾包 ≥ 100
    • 排除區域：廣告活動受眾包 + 網站流量受眾包 ≥ 50
  • 表單顯示規則：
    • 儲存後，會將選擇項目的 path 顯示於「特定族群」區域中
 Audience / Segment 包含：
  • 特定族群
  • 產業鑽石受眾
  • 廣告活動受眾包
  • 網站流量受眾包
排除：
  • 廣告活動受眾包
  • 網站流量受眾包
